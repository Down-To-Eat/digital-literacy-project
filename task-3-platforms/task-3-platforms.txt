In this task, I explored both coding and collaboration platforms to understand how they are used in academic and practical work.

This task helped me learn how to use online platforms effectively for both learning and collaboration. Coding platforms like HackerRank will help me improve my programming skills, while tools like Google Forms will be useful for surveys, quizzes, and academic projects.
